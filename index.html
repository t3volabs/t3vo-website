Frequently Asked Questions

Is my data really secure?

Yes, t3vo uses end-to-end encryption and a zero-knowledge architecture. We never have access to your unencrypted data.

Can I use t3vo on multiple devices?

t3vo supports seamless syncing across all your devices while maintaining top-notch security.

What happens if I forget my master password?

For security reasons, we cannot recover your master password. However, we provide options for secure password recovery using encryption keys.

Is t3vo open-source?

Yes, t3vo is fully open-source. You can review our code on GitHub and even contribute to the project.
